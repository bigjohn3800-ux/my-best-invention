## 1. 시작하기

**마이베스트 발명창업**은 AI 기반 발명 및 창업 교육 플랫폼입니다. 학생들이 AI의 도움을 받아 창의적인 발명 아이디어를 구체화하고, 비즈니스 모델을 설계하며, 투자 피칭 자료를 만들 수 있습니다.

### 주요 기능 요약

| 기능 | 설명 |
|------|------|
| AI 발명 스튜디오 | SCAMPER, TRIZ 기법으로 발명 아이디어 생성 + 특허 명세서 초안 |
| AI 창업 연구소 | 비즈니스 모델 캔버스(BMC) 작성 + IR 피칭덱 생성 |
| 진단 평가 | 창의성/기업가정신 진단 + AI 맞춤 피드백 |
| 학습 과정 | 발명/창업 분야별 단계별 코스 학습 |
| 커뮤니티 갤러리 | 작품 공유, 좋아요, 소통 |
| 대시보드 | 학습 진도, 배지, 프로젝트 관리 |

### 학습 레벨 시스템

플랫폼은 3가지 레벨을 제공합니다. 레벨에 따라 AI 응답의 난이도가 조절됩니다.

- **초급 (Beginner)**: 쉬운 용어와 친절한 설명
- **중급 (Intermediate)**: 전문 용어 포함, 심화 설명
- **고급 (Advanced)**: 전문가 수준의 분석과 기술적 세부사항

## 2. 회원가입 및 로그인

### 회원가입 방법

1. 상단 네비게이션 바에서 **"로그인"** 버튼을 클릭합니다.
2. 로그인 페이지(`/auth`)에서 **"회원가입"** 탭을 선택합니다.
3. 다음 정보를 입력합니다:
   - **아이디** (username): 영문/숫자 조합
   - **비밀번호** (password): 안전한 비밀번호 설정
   - **이름** (displayName): 표시될 이름
4. **"회원가입"** 버튼을 클릭하면 자동으로 로그인됩니다.

### 로그인 방법

1. `/auth` 페이지에서 **"로그인"** 탭을 선택합니다.
2. 아이디와 비밀번호를 입력합니다.
3. **"로그인"** 버튼을 클릭합니다.

### 비회원(게스트) 이용

- 로그인하지 않아도 AI 기능을 **2회까지 무료 체험**할 수 있습니다.
- 체험 후 계속 사용하려면 회원가입이 필요합니다.

## 4. AI 발명 스튜디오

**경로**: `/invention-studio`

AI 발명 스튜디오는 5단계로 구성된 발명 아이디어 개발 도구입니다.

### 단계별 진행

#### 1단계: 기본 정보 입력
- **발명 제목**: 발명품의 이름을 입력합니다.
- **해결할 문제**: 어떤 문제를 해결하고 싶은지 설명합니다.
- **해결 방법**: 어떻게 해결할 것인지 아이디어를 적습니다.
- **대상**: 이 발명이 도움이 될 사람들을 지정합니다.

#### 2단계: SCAMPER 기법
AI가 7가지 SCAMPER 기법을 적용하여 창의적 아이디어를 생성합니다.

| 기법 | 의미 | 예시 |
|------|------|------|
| S (Substitute) | 대체하기 | 재료를 다른 것으로 바꾸면? |
| C (Combine) | 결합하기 | 다른 기능과 합치면? |
| A (Adapt) | 적용하기 | 다른 분야의 아이디어를 가져오면? |
| M (Modify) | 변형하기 | 크기나 모양을 바꾸면? |
| P (Put to other uses) | 다른 용도 | 다른 곳에 쓸 수 있으면? |
| E (Eliminate) | 제거하기 | 불필요한 부분을 빼면? |
| R (Reverse) | 역발상 | 반대로 생각하면? |

- **"SCAMPER 분석 시작"** 버튼을 클릭하면 AI가 실시간으로 아이디어를 생성합니다.

#### 3단계: TRIZ 분석
AI가 TRIZ(창의적 문제 해결 이론)를 적용하여 기술적 모순을 분석하고 발명 원리를 제안합니다.

- **"TRIZ 분석 시작"** 버튼을 클릭합니다.
- AI가 기술적 모순, 물리적 모순, 해결 원리를 제시합니다.

#### 4단계: 아이디어 선택
- SCAMPER와 TRIZ 결과를 검토합니다.
- 마음에 드는 아이디어를 선택하거나 수정합니다.

#### 5단계: 특허 명세서 초안
- AI가 특허 명세서 형식의 문서를 자동 생성합니다.
- 포함 내용: 발명의 명칭, 기술 분야, 배경기술, 해결 과제, 과제 해결 수단, 발명의 효과
- **"명세서 다운로드"** 버튼으로 텍스트 파일을 다운로드할 수 있습니다.

### 프로젝트 저장

- 각 단계에서 **"저장"** 버튼을 눌러 진행 상황을 저장합니다.
- 저장된 프로젝트는 대시보드에서 이어서 작업할 수 있습니다.

### AI 후속 질문

- AI 결과가 나온 후 추가 질문을 할 수 있습니다.
- 결과 아래의 입력창에 질문을 입력하면 AI가 추가 설명을 제공합니다.

## 5. AI 창업 연구소

**경로**: `/startup-lab`

비즈니스 모델 캔버스(BMC) 작성과 IR 피칭덱 생성을 지원합니다.

### BMC(비즈니스 모델 캔버스) 작성

BMC의 9가지 구성 요소를 AI가 도와 작성합니다:

| 번호 | 항목 | 설명 |
|------|------|------|
| 1 | 고객 세그먼트 | 누구를 위한 제품/서비스인가? |
| 2 | 가치 제안 | 어떤 가치를 제공하는가? |
| 3 | 채널 | 어떻게 고객에게 전달하는가? |
| 4 | 고객 관계 | 고객과 어떤 관계를 맺는가? |
| 5 | 수익원 | 어떻게 돈을 버는가? |
| 6 | 핵심 자원 | 무엇이 필요한가? |
| 7 | 핵심 활동 | 무엇을 해야 하는가? |
| 8 | 핵심 파트너 | 누구와 협력하는가? |
| 9 | 비용 구조 | 비용은 얼마나 드는가? |

### 사용 방법

1. **비즈니스 이름**을 입력합니다.
2. BMC 각 칸을 클릭하여 내용을 직접 입력하거나, **AI 어시스트** 버튼으로 AI가 제안하는 내용을 받을 수 있습니다.
3. 모든 칸을 채운 후 **"캔버스 저장"** 버튼으로 저장합니다.

### IR 피칭덱 생성

1. BMC 작성 완료 후 **"IR 피칭덱 생성"** 버튼을 클릭합니다.
2. AI가 투자자 대상 발표 자료를 자동 생성합니다.
3. 포함 내용: 문제 정의, 솔루션, 시장 규모, 비즈니스 모델, 경쟁 우위, 팀, 재무 계획
4. **"다운로드"** 버튼으로 피칭덱을 텍스트 파일로 다운로드할 수 있습니다.

### 프로젝트 관리

- 여러 개의 BMC 프로젝트를 저장/관리할 수 있습니다.
- 이전에 저장한 프로젝트를 불러와 수정할 수 있습니다.

## 6. 진단 평가

**경로**: `/diagnosis`

창의성과 기업가정신을 진단하는 평가 도구입니다.

### 진단 유형

#### 창의성 진단
- **유창성**: 아이디어를 많이 생각해내는 능력
- **독창성**: 새롭고 독특한 아이디어를 내는 능력
- **정교성**: 아이디어를 구체화하는 능력
- **유연성**: 다양한 관점에서 생각하는 능력

#### 기업가정신 진단
- **성장 마인드셋**: 도전과 실패를 배움의 기회로 보는 태도
- **리스크 감수**: 위험을 감수하고 도전하는 자세
- **문제 해결력**: 문제를 찾고 해결하는 능력
- **리더십**: 팀을 이끌고 소통하는 능력

### 진단 방법

1. 진단 유형(창의성 또는 기업가정신)을 선택합니다.
2. 각 항목에 대해 1~10점 사이의 점수를 입력합니다.
3. **"AI 분석 시작"** 버튼을 클릭합니다.
4. AI가 진단 결과를 분석하고 개인 맞춤 피드백을 제공합니다.

### 결과 확인

- 각 항목별 점수와 종합 평가가 표시됩니다.
- AI가 강점과 개선점을 구체적으로 알려줍니다.
- 진단 기록은 대시보드에서 확인할 수 있습니다.

## 7. 학습 과정 (코스)

### 코스 목록

**발명 코스 경로**: `/invention`
**창업 코스 경로**: `/startup`

각 트랙별로 3개의 코스가 준비되어 있습니다 (총 6개).

### 코스 진행 방법

1. 원하는 코스를 선택합니다.
2. **"수강 시작"** 버튼을 클릭하여 등록합니다.
3. 코스 내 단원을 순서대로 학습합니다.
4. 각 단원 완료 시 진도가 자동으로 기록됩니다.
5. 모든 단원을 완료하면 **수료 배지**를 획득합니다.

### 코스 상세 페이지

**경로**: `/course/:id`

- 코스 설명 및 학습 목표
- 단원 목록과 진도 표시
- 학습 자료 및 과제

## 9. 커뮤니티 갤러리

**경로**: `/community`

학생들이 자신의 발명 아이디어나 비즈니스 모델을 공유하는 공간입니다.

### 작품 등록하기

1. **"작품 등록"** 버튼을 클릭합니다 (로그인 필요).
2. 작품 정보를 입력합니다:
   - **유형 선택**: 발명 또는 창업(BMC)
   - **제목**: 작품 제목 입력
   - **설명**: 작품에 대한 설명
   - **태그**: 관련 키워드 (쉼표로 구분)
3. **"등록"** 버튼을 클릭하면 갤러리에 공개됩니다.

### 갤러리 탐색

- **전체**: 모든 작품 보기
- **발명**: 발명 프로젝트만 필터링
- **창업**: BMC 프로젝트만 필터링

### 좋아요 기능

- 마음에 드는 작품에 **하트(좋아요)** 버튼을 누를 수 있습니다.
- 다시 누르면 좋아요가 취소됩니다.
- 각 작품의 좋아요 수가 표시됩니다.

### 작품 삭제

- 자신이 등록한 작품만 삭제할 수 있습니다.
- 작품 카드의 **삭제(휴지통)** 아이콘을 클릭합니다.

## 10. 대시보드

| 코스 완주 | 코스 전체 수료 |

## 11. 수료증

코스를 완료하면 수료증을 발급받을 수 있습니다.

### 수료증 발급 방법

1. 대시보드(`/dashboard`)에서 완료한 코스를 확인합니다.
2. **"수료증 발급"** 버튼을 클릭합니다.
3. 고유 인증 번호가 포함된 수료증이 생성됩니다.
4. 이미지 파일로 다운로드할 수 있습니다.

### 수료증 내용

- 수료자 이름
- 코스명
- 수료일
- 고유 인증 번호

## 12. 플랫폼 가이드

**경로**: `/guide`

플랫폼 사용 방법을 안내하는 가이드 페이지입니다.

## 13. 영감 페이지

**경로**: `/inspiration`

다양한 발명과 창업 아이디어에서 영감을 얻을 수 있는 페이지입니다.

### 기능

- 창의적 아이디어 예시 브라우징
- 카테고리별 아이디어 탐색
- 아이디어를 발명 스튜디오에서 바로 시작할 수 있는 링크

## 14. 프랜차이즈 문의

**경로**: `/franchise`

학교, 학원, 교육 기관을 위한 단체 도입 문의 페이지입니다.

### 문의 방법

1. 프랜차이즈 페이지에서 문의 폼을 작성합니다.
2. 입력 정보:
   - 기관명
   - 담당자 이름
   - 이메일 / 전화번호
   - 예상 인원
   - 문의 내용
3. **"문의 접수"** 버튼을 클릭합니다.
4. 관리자가 확인 후 연락드립니다.

---

## 15. 단체 가입 (초대 코드)

학교나 기관에서 제공받은 초대 코드로 단체에 가입할 수 있습니다.

### 가입 방법

1. 회원가입 시 **초대 코드** 입력란에 코드를 입력합니다.
2. 올바른 코드를 입력하면 해당 단체에 자동으로 소속됩니다.
3. 단체 소속 회원은 교사/멘토의 대시보드에서 진도를 관리받을 수 있습니다.

## 16. 자주 묻는 질문 (FAQ)

**Q: AI 기능을 무료로 사용할 수 있나요?**
A: 비회원은 2회까지 무료 체험이 가능합니다. 회원가입 후에는 제한 없이 사용할 수 있습니다.

**Q: 저장한 프로젝트는 어디서 확인하나요?**
A: 로그인 후 대시보드(`/dashboard`)에서 모든 프로젝트를 확인하고 이어서 작업할 수 있습니다.

**Q: 레벨은 어떻게 바꾸나요?**
A: 대시보드 또는 프로필 설정에서 초급/중급/고급 레벨을 변경할 수 있습니다. 레벨에 따라 AI 응답의 난이도가 달라집니다.

**Q: 특허 명세서를 실제 특허 출원에 사용할 수 있나요?**
A: AI가 생성한 명세서는 초안입니다. 실제 특허 출원을 위해서는 변리사의 검토가 필요합니다.

**Q: 수료증은 공인 인증서인가요?**
A: 플랫폼 자체 발급 수료증으로, 학습 이수를 증명하는 용도입니다.

**Q: 커뮤니티에 올린 작품을 다른 사람이 복사할 수 있나요?**
A: 갤러리에 공개된 내용은 열람 가능하지만, 저작권은 원작자에게 있습니다.
